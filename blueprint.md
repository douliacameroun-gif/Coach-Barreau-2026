# Barreau 2026 Coach

## Overview

This is a React application to help students prepare for the 2026 bar exam. It includes a countdown to the exam date, an interactive quiz, and words of encouragement.

## Tech Stack

*   React
*   Vite
*   Tailwind CSS

## Features

*   **Countdown:** A countdown timer to April 18, 2026.
*   **Interactive Quiz:** A quiz with questions on "Droit Pénal", "Procédure Civile", and "Droit Administratif".
*   **Encouragement Message:** A message to motivate students.

## Design

*   **Theme:** Modern and clean.
*   **Colors:** Green and gold to evoke a legal theme.
*   **Styling:** Tailwind CSS.
